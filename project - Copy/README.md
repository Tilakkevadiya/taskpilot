# Task Pilot – React UI

A simple React web app for tasks, email drafts, meetings, documents, and an assistant. **Frontend only** – no backend. White and blue theme.

## Tech

- **React** + **Vite**
- **React Router**
- **Lucide** icons
- **localStorage** for data (no server)

## Run

```bash
npm install
npm run dev
```

Open **http://localhost:5173** (or the port Vite prints).

## Build

```bash
npm run build
npm run preview
```

## Pages

- **Dashboard** – Overview and recent activity
- **Tasks** – Add and manage tasks (saved in browser)
- **Email** – Compose and save drafts
- **Meetings** – Schedule and list meetings
- **Documents** – List and manage documents
- **Assistant** – Chat and voice commands (browser APIs)

All data stays in your browser (localStorage). No backend or API.

## Theme

White background with blue accents (`#2563eb`).
